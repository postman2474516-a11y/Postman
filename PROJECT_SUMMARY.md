### 推荐部署方案

#### 1. GitHub Pages（免费）
```bash
# 1. 创建GitHub仓库
# 2. 推送代码
# 3. 在Settings > Pages中启用
# 4. 访问：https://yourusername.github.io/redcoast-website/
```

#### 2. Netlify（免费，支持自定义域名）
```bash
# 1. 拖拽文件夹到Netlify
# 2. 自动部署
# 3. 配置自定义域名
```

#### 3. Vercel（免费，性能优秀）
```bash
# 1. 连接Git仓库
# 2. 自动部署
# 3. 配置自定义域名
```
